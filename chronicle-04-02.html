Triplicata typeform

Processing
Processing is one of the most direct methods to construct generative design. Parameters such as size, direction and distance create restrictions in which the generative design can be constructed. Processing has potential for wider implimentation in web and interactive space. Processing can function through a number of principle methods, including drawing and agent functions. Processing can also use typography in 2D and 3D to construct alternative, generative ways of experiencing language.
