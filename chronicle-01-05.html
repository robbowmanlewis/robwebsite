Three-dimensional grid design
Grid design in 3D requires attention towards depth of imagery and functions of perspective. This grid uses asymmetry over a single page, progressing to symmetry over a full page. This allows for a building-block style customisation, allowing designs to be informed/executed in a myriad of ways whilst always holding structure. Three-dimensional grid design can be applied to new technologies which are based on 3D frameworks.
